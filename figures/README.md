Вставка в Word: **Вставка → Рисунки → это устройство** → выбрать `.svg` (Word 2016+ и новее).

Если нужен только PNG: открой `.svg` в браузере → скрин/печать в PDF, либо онлайн-конвертер SVG→PNG.

### Как согласовать с уже существующими «Рисунок 1–8» в уп 02

В отчёте ПМ.02 уже заняты номера под другие темы. Эти четыре файла — **отдельные** диаграммы под требования **IDEF0 / DFD / UML**. Варианты:

1. Вставить их в **приложение** как **Рисунок А.1 … А.4** (если по методичке приложения нумеруются отдельно).
2. Либо сдвинуть нумерацию в основной части (тогда правь подписи вручную под требования преподавателя).
