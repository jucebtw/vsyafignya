# Отчёт «Титульник Часть 2» — обновлена только 2-я часть

## Что сделано

- Файл **`Титульник Часть 2 — обновлена только 2-я часть.docx`** собран так:
  - **Часть 1** (титульный блок, содержание, введение, сведения о практике, раздел **1. Основная часть** 1.1–1.6) взята из исходного **`Титульник Часть 2.doc`** **без изменения текста** (восстановление строк из документа).
  - **Часть 2** (раздел **2. Индивидуальное задание** с подпунктами **2.1–2.12**), а также **Заключение** и **Список использованных источников** сформированы на основе текста **`уп 02 2 часть  (2).docx`**, с дополнительным развёрнутым содержанием по теме АСУ складом, оформлением по требованиям **`Методичка для ПП.docx`** (шрифт Times New Roman 14, интервал 1,5, выравнивание по ширине, абзац 1,25 см).

- Исходные файлы **не перезаписывались**.

## Скрипт пересборки

При необходимости правок запустите:

```bash
python3 build_titulnik_part2_merge.py
```

## Примечание

После открытия в Word рекомендуется: **обновить поля оглавления** (если используете автособираемое содержание), проверить **сквозную нумерацию страниц** и вставить **рисунки** (в исходном ПМ.02 они ссылаются в тексте — при необходимости перенесите из отчёта по практике).
