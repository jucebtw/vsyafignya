# Отчёт «Титульник Часть 2» — обновлена только 2-я часть

## Что сделано

- **`Титульник Часть 2 — обновлена только 2-я часть.docx`**
  - **Часть 1** — текст из **`Титульник Часть 2.doc`** до раздела «2. Индивидуальное задание». Оглавление **пересобрано без полей Word** (без «битых» HYPERLINK/PAGEREF в тексте).
  - **Часть 2** (2.1–2.12), **Заключение**, **Список** — по **`уп 02 2 часть  (2).docx`**, текст **сжат** (без лишней «воды»), ориентир **~50 страниц** всего документа.
  - **Таблицы 1–2** (трудозатраты, сравнение типов систем): название над таблицей, оформление по методичке.
  - **Рисунки 1–8** перенесены из отчёта ПМ.02 (встроенные изображения), с подписями как в исходнике; в тексте есть ссылки на блоки иллюстраций.
